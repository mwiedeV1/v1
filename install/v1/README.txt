V1 Scripting Language V 0.96
============================

start_editor.sh will start a Webserver on localhost port 8080.
http://localhost:8080/editor show the editor with V1 examples in your Web browser.
To change the HTTP port please edit the file app/webserver.v1 and change constant SERVER_PORT=8080


See also: https://www.v1-script.net
